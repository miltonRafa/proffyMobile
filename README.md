# Proffy Mobile

Aplicativo mobile do projeto Proffy - plataforma de estudos online.

## Tecnologias

- Android
- Kotlin
- Gradle

## Como executar

1. Clone o repositório
2. Abra o projeto no Android Studio
3. Execute o projeto
